Package io.spine.annotation

See: Description

  • Annotation Summary 
    Annotation Description
    BetaSignifies that a public API is subject to incompatible changes, or even removal in a future release.
    ExperimentalIndicates a public API that can change at any time and has no guarantee of API stability and backward-compatibility.
    GeneratedThe Generated annotation is used to mark source code that has been generated.
    GeneratedMixinIndicates an interface which is implemented in the generated code adding desired behavior using default methods declared in the interface.
    InternalMarks a program element (class, method, etc.) as internal to a module or a bounded context, indicating it is NOT part of the public API and hence not intended for use outside the module's development team.
    ModifiedThe Modified annotation is used to mark the source code which was created by one code generator and then modified by another code generator hereby referenced as "modifier".
    SPIAnnotates a program element (class, method, package, etc.) as an element of Service Provider Interface (SPI).
    TestOnlyAnnotates a program element that is intended to be used only in test code.
    VisibleForTestingAnnotates a program element that exists, or is more widely visible than otherwise necessary, only for use in test code.
