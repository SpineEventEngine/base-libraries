      • withMask

        @CanIgnoreReturnValue() final B withMask(Array<SubscribableField> fields)

        Applies the paths of the passed fields as a field mask to each of the resulting records.

        If the mask is not set, the query results contain the records as-is.

        Any previously set mask values are overridden by this method call.

        Returns:

        this instance of query builder, for chaining
