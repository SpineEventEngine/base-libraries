        @Deprecated() boolean isRejections()

        Returns true if the source file matches conventions for rejection files.

        A valid rejections file must:

        • be named ending on "rejections.proto";
        • have the java_multiple_files option set to false;
        • either have a java_outer_classname value which ends with isValidOuterClassName “Rejections”}, or not have the java_outer_classname option set at all.
