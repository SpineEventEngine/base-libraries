      • isId

         boolean isId()

        Determines whether the field is an ID.

        An ID satisfies the following conditions:

        • Declared as the first field. The "first" field is determined by the order in which fields appear in the Protobuf message definition (reading order from top to bottom), not by the field number.
        • Declared inside an entity state message or a command message;
        • Is not a map or a repeated field.
        Returns:

        true if the field is an entity ID, false otherwise
