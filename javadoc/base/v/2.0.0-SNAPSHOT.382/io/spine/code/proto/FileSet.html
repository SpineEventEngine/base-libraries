      • parseAsKnownFiles

         static FileSet parseAsKnownFiles(File descriptorSet)

        Parses the given descriptor set file and resolves the descriptors from KnownTypes.

        The read descriptors are intentionally not linked. Instead, the descriptors with such names are found in the KnownTypes and returned as a file set. This way, all the dependencies get resolved.

        If some of the parsed files are not found in the known types, an IllegalStateException is thrown.

        Parameters:
        descriptorSet - the descriptor set file to parse
        Returns:

        new file set
