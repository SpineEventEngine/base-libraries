Package io.spine.io

This package provides utilities specific to enable Spine working with code files in code generation, project structure analysis, and other tasks that are not covered by well-known file management libraries.

For more file-related utilities, please see:

  • Files from NIO
  • Paths from NIO2
  • Files from Guava
  • Kotlin I/O extensions from Utils.kt
