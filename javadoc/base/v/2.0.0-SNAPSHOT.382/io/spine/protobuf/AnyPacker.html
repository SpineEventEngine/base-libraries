Package io.spine.protobuf

Class AnyPacker

  • All Implemented Interfaces:
    
public final class AnyPacker

    Utilities for packing messages into Any and unpacking them.

    When packing, the AnyPacker takes care of obtaining correct type URL prefix for the passed messages.

    When unpacking, the AnyPacker obtains a Java class matching the type URL from the given instance of Any.
