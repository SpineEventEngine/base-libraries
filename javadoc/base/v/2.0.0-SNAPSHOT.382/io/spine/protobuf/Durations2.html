Package io.spine.protobuf

Class Durations2

  • All Implemented Interfaces:
    
public final class Durations2

    Utility class for working with durations in addition to those available from the Durations class in the Protobuf Util library.

    Use import static io.spine.protobuf.Durations2.* for compact initialization like this: 

    
    Duration d = add(hours(2), minutes(30));

         static Duration add(@Nullable() Duration d1, @Nullable() Duration d2)

        Adds two durations one of which or both can be null.

        This method supplements the utility from Protobuf Utils for accepting nulls.

        Parameters:
        d1 - a duration to add, could be null
        d2 - another duration to add, could be null
        Returns:
        • sum of two durations if both of them are non-null
        • another non-null value, if one is null
        • ZERO if both values are null
