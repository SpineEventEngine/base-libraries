        Obtains a leading comments by the LocationPath.

        This method logs a warning message if the file declaring this message type does not have source code info produced ty Protobuf compiler. The warning message provides instructions for configuring the Protobuf Gradle plugin for enabling source code info.

        Parameters:
        locationPath - the location path to get leading comments
        Returns:

        the leading comments or empty Optional if there are no such comments or a descriptor was generated without source code information
