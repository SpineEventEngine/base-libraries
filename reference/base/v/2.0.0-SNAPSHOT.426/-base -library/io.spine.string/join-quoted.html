joinQuoted

fun Iterable<Placeholder>.joinQuoted(): String(source)

Joins all placeholders in this iterable into a single string, separating them with commas and wrapping each placeholder name in backticks.

This is a convenience method for formatting lists of placeholders in error messages and diagnostic output.

For example, given placeholders with names "foo", "bar", and "baz", this method returns:

`foo`, `bar`, `baz`.
