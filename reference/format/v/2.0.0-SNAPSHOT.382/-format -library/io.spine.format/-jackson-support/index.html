JacksonSupport

@SPI
abstract class JacksonSupport(source)

An abstract base class for I/O operations using the Jackson library.

The primary subclasses of this class are the abstract types JacksonWriter and JacksonParser, which provide the foundation for reading from and writing to various data formats.

To support a new data format beyond the existing Formats, you will typically define a pair of classes extending JacksonWriter and JacksonParser, rather than subclassing this base class directly.

Adding Support for a New Format

  1. Create a new writer class by extending JacksonWriter.

  2. Create a corresponding parser class by extending JacksonParser.

  3. Define an object that extends Format, and provide instances of the new writer and parser, along with the appropriate file extension(s).

  4. If you are a contributor to the Spine SDK, nest the new format object under the Format class. This helps maintain an enumeration-like structure. We avoid using a Kotlin enum here because we need generic parameters. Also, be sure to add the new object to the Format.entries list.
